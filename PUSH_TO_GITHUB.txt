1. Create a new GitHub repository named:
   fruit-classification-cnn

2. Put these files into your local project folder:
   - README.md
   - PORTFOLIO_SUMMARY.md
   - PROJECT_STRUCTURE.md
   - data_README.md
   - .gitignore
   - requirements.txt
   - environment.yml
   - notebooks/Fruit_Classification_Assessmennt-1844721.ipynb
   - model/fruit_classifier_final_best_model.keras
   - src/predict.py
   - src/demo_inference.py

3. Open terminal in that folder and run:

   git init
   git add .
   git commit -m "Initial commit - fruit classification CNN portfolio project"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/fruit-classification-cnn.git
   git push -u origin main

4. After pushing, check GitHub and make sure:
   - README renders correctly
   - notebook opens
   - model file is visible
   - no huge raw dataset folders were uploaded by mistake
